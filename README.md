# App-Camera

### Functionalities:

- Catch
- Front and rear camera
- Flash
- zoom
- Focus

### Installed libraries:

- npx expo install expo-camera expo-contacts expo-sensors
- npx expo install expo-camera
- npx expo install @react-native-community/slider 
- npx expo install expo-media-library
